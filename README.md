MediTrack – Medicine Expiry Tracker

MediTrack is a web-based medicine management system that helps users track and manage their medicines efficiently. The application allows users to store medicine details, monitor expiry dates, and receive visual alerts when medicines are near expiry or expired.

The system is designed with a clean user interface, secure login system, and personalized dashboards, ensuring that each user can manage their medicines independently.

🚀 Features

👤 User Authentication

Secure login and registration system

Each user has private access to their own medicines

Session-based authentication using Flask

💊 Medicine Management

users can:

Add new medicines

View all stored medicines

Edit medicine details

Delete medicines

🔎 Search Function

Search medicines by name for quick access
